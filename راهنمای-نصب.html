تبریک! اپلیکیشن شما آماده است

اکنون اپ روی هاست رایگان GitHub قرار دارد و از هر دستگاهی قابل دسترسی است. هر بار که نیاز به به‌روزرسانی داشتید، فقط فایل‌ها را در همان Repository جایگزین کنید.

سوالات متداول
❓ آیا داده‌های من خصوصی می‌ماند؟
بله. حتی اگر Repository شما Public باشد، داده‌هایی که شما در اپ وارد می‌کنید (وزن، یادداشت، مکمل‌ها و...) فقط روی گوشی خودتان ذخیره می‌شود. هیچ سرور و دیتابیسی در کار نیست.
❓ چطور اپ را به‌روزرسانی کنم؟
به Repository خود بروید، روی فایل مورد نظر کلیک کنید، آیکون مداد ✏️ را بزنید، تغییرات را اعمال کنید و Commit کنید. یا فایل‌ها را با Upload جایگزین کنید.
❓ آیا می‌توانم نام آدرس را عوض کنم؟
بله. در Settings → General می‌توانید نام Repository را تغییر دهید. مثلاً اگر نام را به health تغییر دهید، آدرس می‌شود: YOUR-USERNAME.github.io/health/
❓ اگر بعداً خواستم تغییرش بدهم چی؟
به‌من بگو چی می‌خواهی، تغییرات را برات اعمال می‌کنم و فقط فایل index.html را روی GitHub جایگزین می‌کنی.
❓ محدودیت پهنای باند یا تعداد بازدید دارد؟
GitHub Pages تا ۱۰۰ گیگابایت پهنای باند ماهانه و ۱۰۰ گیگابایت حجم رایگان می‌دهد - برای استفاده شخصی بیش از کافی است.
❓ اگر مشکلی پیش آمد چه کنم؟
به Settings → Pages برگرد و مطمئن شو که Source روی main branch تنظیم است. اگر هنوز کار نکرد، حدود ۵-۱۰ دقیقه صبر کن (گاهی زمان می‌برد).
